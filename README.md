# MLCS

The Implementation gives the theoretical justification of the method discussed in the paper below.
# Running command

```bash
gcc _MLCS_.cpp -lstdc++

./a.out
```
# Input 
[Random DNA string generator](http://www.bioinformatics.org/sms2/random_dna.html)
